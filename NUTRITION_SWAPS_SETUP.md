# 🍎 הוראות הוספת טבלת המרות התזונה המלאה

## 📋 מה הקובץ הזה עושה?

הקובץ `migration-add-complete-nutrition-swaps.sql` מוסיף את כל המזונות והמרות לפי הטבלה שסיפקת:

- 🍚 **פחמימות**: אורז, פסטה, תפוח אדמה, בטטה, קוסקוס, קינואה, עדשים, כוסמת, בורגול
- 🍗 **חלבונים**: פרגית, חזה עוף, הודו, דגים, טונה, קוטג', גבינות, פסטרמה, ביצים, אבקת חלבון
- 🥑 **שומנים**: טחינה, אגוזים, שקדים, בוטנים, זיתים, אבוקדו, שמן זית, חומוס, במבה, שוקולד
- 🥖 **לחם/פיתות**: פיתה, לחם לבן, לחמנייה, פריכייה, חלה

## 🚀 איך להריץ?

### שלב 1: פתח Supabase Dashboard
1. לך ל-https://app.supabase.com
2. בחר את הפרויקט שלך
3. לחץ על **SQL Editor** בתפריט השמאלי

### שלב 2: הרץ את המיגרציה
1. פתח את הקובץ `migration-add-complete-nutrition-swaps.sql`
2. **העתק את כל התוכן** (Ctrl+A, Ctrl+C)
3. הדבק ב-SQL Editor
4. לחץ על **Run** (או Ctrl+Enter)

### שלב 3: בדיקה
1. לך ל-**Table Editor** ב-Supabase
2. בחר את הטבלה `nutrition_swaps`
3. תראה את כל המזונות החדשים!

## ⚠️ הערות חשובות:

1. **הסקריפט משתמש ב-`ON CONFLICT`** - אם מזון כבר קיים, הוא יעודכן במקום להוסיף כפילות
2. **אם אין UNIQUE constraint** - הסקריפט יוסיף אותו אוטומטית
3. **הנתונים מבוססים על הטבלה שסיפקת** - כל המרות והמשקלים מחושבים נכון

## 📊 מה כל מזון כולל?

כל מזון כולל:
- `food_name` - שם המזון
- `category` - קטגוריה (carbs/protein/fat/bread)
- `conversion_factor` - גורם המרה (יחסית למזון הבסיס בקטגוריה)
- `protein_per_100g` - חלבון ל-100 גרם
- `carbs_per_100g` - פחמימות ל-100 גרם
- `fat_per_100g` - שומן ל-100 גרם
- `calories_per_100g` - קלוריות ל-100 גרם

## ✅ אחרי ההרצה:

לאחר הרצת הסקריפט, המתאמן יוכל:
- לראות את כל המזונות ברשימה
- לבצע המרות בין מזונות שונים
- להשתמש במזונות מהתפריט שהמאמן הגדיר
